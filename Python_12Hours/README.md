# Python-Programming

## Overview

This repository contains my work from following the “Python Tutorial for Beginners – Full Course 2024” video course. The purpose of this repository is to document my progress while learning Python fundamentals, practicing core programming concepts, and completing a range of small beginner-friendly projects.

The course covers essential Python topics such as variables, user input, conditionals, loops, collections, functions, object-oriented programming, file handling, exceptions, decorators, and more. In addition to theory and syntax, it also includes several practical mini-projects that helped reinforce the concepts through hands-on coding.

This repository serves as both a learning log and a reference collection of Python examples and exercises.

### What I Covered

Through this course, I worked through topics including:
- variables and type casting
- user input and string handling
- arithmetic and conditional logic
- loops and nested loops
- lists, sets, tuples, dictionaries, and 2D collections
- functions, default arguments, keyword arguments, *args and **kwargs
- iterables, membership operators, and list comprehensions
- modules, scope, and if __name__ == "__main__"
- object-oriented programming
- inheritance, polymorphism, duck typing, and magic methods
- decorators and properties
- exception handling
- file reading and writing
- dates and times
