# Python-Programming

A collection of Python practice files, exercises, and small projects created while learning Python fundamentals and object-oriented programming.

This repository combines work from several Python learning resources, including beginner Python courses, OOP-focused tutorials, and my own experiments and notes.

⸻

## Overview

The repository is organized into three main parts:

.
├── Python_12Hours/   # Beginner Python course with many small scripts and projects
├── Python_1Hour/     # Short object-oriented programming course
├── Python_OOP/       # Additional OOP practice and examples
└── README.md

⸻

## Main Folders

Folder <--> Description
Python_12Hours <--> Work from the Bro Code 12-hour Python course, covering Python fundamentals, functions, collections, file handling, exceptions, OOP, and small projects.
Python_1Hour <--> OOP practice based on a shorter Tech With Tim-style course, focusing on classes, objects, inheritance, class methods, static methods, and combining objects.
Python_OOP <--> Additional object-oriented programming practice with classes, attributes, methods, protected fields, object composition, and simple class design.
